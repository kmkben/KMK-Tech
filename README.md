# KMK-Tech – Personal Tech Portfolio

KMK-Tech is my personal technology portfolio where I share projects, articles, and insights related to software engineering, embedded systems, IoT, and cybersecurity.

The goal of this platform is to document my technical journey, showcase my work, and share knowledge with the developer and security communities.

## Installation

Clone the repository:

```
git clone https://github.com/kmkben/KMK-Tech.git
```

Navigate into the project directory:

```
cd KMK-Tech
```

Create a virtual environment:

```
python -m venv venv
```

Activate the environment:

Mac/Linux

```
source venv/bin/activate
```

Windows

```
venv\Scripts\activate
```

Install dependencies:

```
pip install -r requirements.txt
```

Apply migrations:

```
python manage.py migrate
```

Run the development server:

```
python manage.py runserver
```
